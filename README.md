# FocusPath: AI-Powered Study & Task Planner

## Project Overview
FocusPath is an intelligent scheduling assistant designed to help students optimize their study sessions. It uses Machine Learning to categorize task priority and Search Algorithms to manage complex project dependencies.

## Features (Planned)
- **Priority Classifier:** A Machine Learning model to predict task urgency and burnout risk.
- **Dependency Resolver:** Using Graph-based algorithms (DFS/BFS) to find the best order of operations.
- **Data-Driven Insights:** Preprocessing student task data for better time management.

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn (ML), Matplotlib (Visualization)
- **Algorithms:** Decision Trees, Depth-First Search (DFS)
