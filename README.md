# FocusPath: AI-Powered Study & Task Planner 🚀

An intelligent scheduling assistant that uses Machine Learning to prioritize student tasks and Graph Algorithms (DFS) to resolve complex study dependencies.

## 📌 Problem Statement
Students often suffer from "decision paralysis" when managing multiple subjects. Traditional to-do lists don't account for task difficulty or the logical order of learning (e.g., you can't "Write Code" before "Learning Theory"). FocusPath solves this by using AI to build an optimized study path.

## 🛠️ Features
- **ML Priority Classifier:** Uses a **Decision Tree** model to predict task urgency (High/Medium/Low) based on difficulty and deadlines.
- **Dependency Resolver:** Implements **Depth-First Search (DFS)** to perform a topological sort, ensuring all prerequisites are met.
- **Automated Scheduling:** Combines ML and Search logic to generate a step-by-step study plan.

## 💻 Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, Scikit-learn (ML), Matplotlib
- **Algorithms:** Supervised Learning (Classification), Depth-First Search (DFS)
